# Analisis-de-performance---Ford-Ka
Este proyecto consiste en el desarrollo de un dashboard interactivo en Power BI para analizar el rendimiento, mantenimiento y comportamiento operativo de un Ford Ka a partir de datos reales del vehículo.

Características
📊 Dashboard interactivo desarrollado en Power BI.
📈 Análisis de tendencias y métricas del vehículo.
⛽ Seguimiento del consumo de combustible.
🔧 Control del mantenimiento realizado.
📅 Análisis temporal de los datos.
📌 Indicadores clave (KPIs).
🎛️ Filtros dinámicos para explorar la información.
