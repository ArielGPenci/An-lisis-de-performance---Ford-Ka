
# 🚗 Analisis de performance Ford Ka - Dashboard

## 📖 Descripción

Este proyecto consiste en el desarrollo de un **dashboard interactivo en Power BI** para analizar el rendimiento, mantenimiento y comportamiento operativo de un Ford Ka a partir de datos reales del vehículo.

El objetivo es transformar datos en información útil mediante visualizaciones claras, indicadores clave (KPIs) y filtros interactivos que faciliten la toma de decisiones y el seguimiento del estado del automóvil.

Este proyecto forma parte de mi **portfolio de Analista de Datos**, demostrando habilidades en Business Intelligence, modelado de datos, visualización y desarrollo de dashboards profesionales.

## 🎯 Objetivos del proyecto

- Analizar el rendimiento general del vehículo.
- Visualizar indicadores clave mediante KPIs.

- 🚗 Kilometraje total
